# Tutorial Images

这个目录用于存放 README 图文教程中的截图。

## 使用方式

- 直接用你的截图替换下面同名文件即可。
- 如果你想改成 `png` 或 `jpg`，同时把仓库根目录 `README.md` 里的对应图片路径后缀一起改掉即可。
- 建议截图宽度保持在 `1400px` 左右，方便 GitHub README 展示。

## 文件对应关系

| 文件名 | 建议内容 |
| --- | --- |
| `01-open-extension-popup.svg` | 浏览器中打开扩展配置页 |
| `02-save-captcha-settings.svg` | 填写并保存超级鹰配置 |
| `03-open-hdyy-page.svg` | 打开讲座预约页面 |
| `04-click-inline-schedule-button.svg` | 在讲座卡片旁点击提前预约 |
| `05-open-task-sidebar.svg` | 打开侧边栏查看任务 |
| `06-check-task-status.svg` | 查看任务状态和结果 |
| `07-cancel-task.svg` | 在侧边栏取消任务 |
