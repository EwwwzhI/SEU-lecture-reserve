# 东南大学讲座预约助手（SEU Lecture Reserve Assistant）

面向东南大学学生的 Chrome 扩展，用于在讲座预约页面识别活动、创建预约任务，并在到点后自动执行预约请求。

## 1. 项目定位

- 目标：减少手动盯页面和重复点击，提高讲座预约成功率与操作效率。
- 形态：浏览器侧边栏 + 后台定时任务。
- 当前支持页面（固定）：
  - `https://ehall.seu.edu.cn/gsapp/sys/yddjzxxtjappseu/*default/index.do#/hdyy`

## 2. 功能特性

- 自动识别当前页面中的讲座活动。
- 支持“提前预约任务”（活动未开放时先排队，到时自动提交）。
- 后台自动重试提交（可配置间隔与最大次数）。
- 接入超级鹰验证码识别服务。

## 3. 安装方式

1. 下载并解压项目代码。
2. 打开 Chrome，进入 `chrome://extensions/`。
3. 开启右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目的 `Assistant` 目录。

## 4. 快速开始

1. 点击扩展图标，打开配置页。
2. 填写验证码服务账号信息（见“接口账户配置”）。
3. 建议保留默认重试参数，点击“保存配置”。
4. 打开目标页面：`#/hdyy`。
5. 在右侧“东南大学讲座预约”侧边栏中选择活动并设置触发时间。
6. 在“计划任务”中观察任务状态，必要时可取消或重试。

## 5. 接口账户配置（超级鹰）

由于扩展需要自动获取并识别验证码，默认使用超级鹰接口服务。

1. 访问 `http://www.chaojiying.com/`，注册账号并充值（通常 1 元即可测试）。
2. 在个人中心申请“软件 ID（softid）”。
3. 将以下信息填写到扩展配置页对应字段：
   - 用户名 -> `用户`
   - 密码 -> `密码`
   - 软件 ID -> `软件ID`

## 6. 配置项说明

| 配置项           | 字段名               | 说明               | 推荐值   |
| ---------------- | -------------------- | ------------------ | -------- |
| 超级鹰用户名     | `captchaUser`      | 验证码平台账号     | 必填     |
| 超级鹰密码       | `captchaPass`      | 验证码平台密码     | 必填     |
| 软件 ID          | `captchaSoftid`    | 超级鹰软件 ID      | 必填     |
| 验证码类型       | `captchaCodetype`  | 超级鹰验证码类型   | `1902` |
| 重试间隔（毫秒） | `retryIntervalMs`  | 每次预约请求间隔   | `500`  |
| 最大重试次数     | `retryMaxAttempts` | 单任务最大尝试次数 | `120`  |

## 7. 常见问题

### Q1：侧边栏没有活动

- 确认当前 URL 是 `#/hdyy` 页面。
- 刷新页面后重试。
- 确认扩展已更新到最新代码并重新加载。

### Q2：任务创建成功但没有抢到

- 确认预约时账号仍处于登录状态。
- 检查验证码账户余额是否充足、`softid` 是否正确。
- 适当提高 `retryMaxAttempts` 或降低 `retryIntervalMs`。

### Q3：点击取消任务时报扩展上下文错误

- 这是扩展热更新后的常见现象，刷新页面并重新打开侧边栏即可。

## 8. 注意事项

- 本工具仅辅助提交，不保证 100% 成功。
- 预约结果受平台开放时间、名额、网络与账号状态等因素影响。
- 请遵守学校平台使用规范，合理使用自动化功能。
