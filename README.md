# 东南大学讲座预约助手 (SEU Lecture Reserve Assistant)

这是一个专为东南大学学生设计的 Chrome 浏览器扩展程序，旨在简化讲座预约流程，通过自动化定时任务帮助用户更轻松地抢到热门讲座。

## 主要功能

- **自动化识别**：在东南大学预约平台页面自动识别可预约的讲座信息。
- **定时预约**：用户可自行设定预约时间，扩展将在后台自动执行预约请求。
- **优雅界面**：采用独特的“纸张质感”测栏设计，视觉体验优雅且专业。
- **自适应缩放**：UI 完美适配从 1080P 到 4K 的各种屏幕分辨率。
- **独立任务管理**：检出的讲座列表与已计划的任务列表独立滚动，管理高效。

## 安装说明

1. 下载并解压本项目代码。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，并选择 `seu-lecture-reserve-assistant` 目录。

## 如何使用

### 1. 全局配置 (便签本)
- 点击浏览器工具栏中的插件图标，打开 **“配置便签”**。
- 在便签中输入必要的全局参数（如验证信息或辅助配置）。
- 系统会自动保存您的输入，确保后台预约任务能够顺利执行。

### 2. 页面交互 (侧边栏)
- 登录东南大学讲座预约平台。
- 页面右侧会出现一个蓝色标签 **“东南大学讲座预约”**。
- 点击标签滑出“纸质笔记本”样式的侧边栏。
- 在侧边栏中查看当前页面的讲座，输入预约时间并点击“预约”即可创建任务。
- 您可以在“计划任务”区域查看进度或取消已排期的任务。

## 注意事项

- 请确保在预约时间到达时，浏览器处于开启状态且网络连接正常。
- 本工具仅为辅助预约，不保证 100% 成功，请务必关注官方预约结果。
